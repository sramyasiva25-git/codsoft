#  Password Generator
##  Description

This project is a simple password generator developed using Python.
It generates a random and secure password based on the length entered by the user, using a combination of letters, numbers, and symbols.

##  Features

  - Generates random passwords

  - User can choose password length

  - Uses letters, digits, and symbols

  - Simple and easy-to-use console program

##  Technologies Used

  - Python
